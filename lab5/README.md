## Запуск проекта
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Tart1lya/labs_asd.git
   ```
2. Перейдите в папку с проектом:
   ```bash
   cd labs_asd/lab5
   ```
3. **Запуск всех задач**
   ```bash
    for script in lab*/src/*.py; do PYTHONPATH=$(pwd) python "$script"; done
   ```

## Тестирование
